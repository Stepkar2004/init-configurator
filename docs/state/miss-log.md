# Miss-log

> One line per event: a task arrived that matched NO skill and got free-styled.
> Format: `date · what the task was · what was done instead`.
> The skill-manager triages: an entry recurring twice has earned a home (procedure in an
> existing skill, or a new skill). An unlogged miss is the system deciding it is
> complete — log first, keep working.
