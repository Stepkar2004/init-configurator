# Roadmap

> Derived from `../vision.md`; re-derived when the vision changes. Chunks, in order.

1. **Dogfood the genome on a real bootstrap.** Use the `bootstrap` skill to phase-0 a
   real downstream project (the climbing-log example or a job-relevant one). Every
   friction point becomes an `evolve` diff — the skills' first real evolution.
2. **Publish prep.** Remote at github.com/Stepkar2004/init-configurator, README
   screencast/GIF of describe → doctor → run, LinkedIn reflection draft (ship visibly).
3. **`initc spawn` / `initc absorb` as commands.** The procedures live in the `absorb`
   skill today; promote them to CLI once they've been executed by hand at least once
   and the shape is proven.
4. **Rot-check pass #1** (scheduled, not reactive): watch every gate fail, stale-pin
   hunt over the references.

Parked / undecided:

- The generalized system (Part II of `../design/agentic-base.md`) — separate project,
  possibly closed-source. TheBrain2 Law 11 (job-first) holds the veto on when.
